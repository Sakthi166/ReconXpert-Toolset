Tools Description

Nmap

A network scanning tool used for discovering hosts and services on a network.

Usage: nmap [options] [target]
Example: nmap -sS 192.168.1.1

Nmap Vulnerability Scanning

Nmap's vulnerability scanning feature allows you to identify potential vulnerabilities in the target system.

Usage: nmap -sV --script vuln [target]
Example: nmap -sV --script vuln 192.168.1.1

Hydra

Hydra is a A fast brute force login cracker for various protocols.

Usage: hydra -l [username] -P [password_list] [target] [service]
Example: hydra -l admin -P passwords.txt 192.168.1.1 ssh

John the Ripper

John the Ripper is a password cracking tool. It can be used to perform dictionary, brute force, and hybrid attacks on various password hashes.

Usage: john [hash_file] --wordlist=[wordlist_file]
Example: john hash.txt --wordlist=rockyou.txt

WPScan

WPScan is used for identifying security issues in WordPress websites, such as vulnerable plugins, themes, and weak credentials.

Usage: wpscan --url [target]
Example: wpscan --url http://example.com

Sublist3r

Sublist3r is a tool used to gather subdomains of a target domain.

Usage: sublist3r -d [domain]
Example: sublist3r -d example.com

TheHarvester

TheHarvester is a tool used to gather emails, subdomains, hosts, and employee names from various public sources.

Usage: theharvester -d [domain] -b [source]
Example: theharvester -d example.com -b google

DNSRecon

DNSrecon is a tool used to perform DNS reconnaissance and subdomain enumeration.

Usage: dnsrecon -d [domain]
Example: dnsrecon -d example.com

Wafw00f

Wafwoof is a tool used to fingerprint web application firewalls (WAFs) and determine their version.

Usage: wafw00f [url]
Example: wafw00f http://example.com

NExtexc

NExtexc is a tool used to extract links and emails from web pages.

Usage: nextexc [options] [target]
Example: nextexc -d example.com

Nslookup

Nslookup is a command-line tool used to query DNS servers for information about domain names and IP addresses.

Usage: nslookup [domain]
Example: nslookup example.com

Nikto

Nikto is a tool used to perform web server scanning and vulnerability testing.

Usage: nikto -h [target]
Example: nikto -h http://example.com

Gobuster

Gobuster is a tool used to brute force directories and files on web servers.

Usage: gobuster dir -u [url] -w [wordlist]
Example: gobuster dir -u http://example.com -w common.txt

WhatWeb

Whatweb is a tool used to identify web technologies and version information.

Usage: whatweb [target]
Example: hatweb http://example.com

DNSenum

DNSenum is a tool used to perform DNS enumeration and gather information about a target domain.

Usage: dnsenum [domain]
Example: dnsenum example.com

Nmap Basic Script Scan

Nmap's script scanning feature allows you to run pre-defined scripts against a target system to identify potential vulnerabilities.

Usage: nmap -sC [target]
Example: nmap -sC 192.168.1.1

Check Live Hosts

Nmap's ping scan feature allows you to identifies active hosts in a network.

Usage: nmap -sn [target_range]
Example: nmap -sn 192.168.1.0/24

Msfconsole port scan

Metasploit's msfconsole allows you to perform port scanning and vulnerability testing.

Usage: msfconsole -q -x use auxiliary/scanner/portscan/tcp; set RHOSTS [target]; run
Example: msfconsole -q -x "use auxiliary/scanner/portscan/tcp; set RHOSTS 192.168.1.1; run
